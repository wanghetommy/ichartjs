自定义组件接口类。可以实现自己扩展到小组件。
需要注意的是自定义的组件应该在绘图区域里进行绘制。

此组件的this.variable.animation是一个对象变量。此对象中有3个属性:
animating:是否处于动画中
time : 起始时间
